Botón¶

Uso¶

Utiliza la palabra clave Botón para cualquier elemento interactivo en la página:

Página "Registrarse"
Bloque "Datos personales"
    Campo "Nombre"
    Campo "Apellidos"
    Campo "correo electrónico"
    Botón "Registrarse"

Igual que con otras palabras claves el orden de palabras y la capitalización son opcionales:

...
"Registrarse" botón
"Entrar" botón

Ejemplos¶

Botones en un asistente de instalación:

Página "Asistente de instalación"
Bloque "Ajustes"
    "Aquí van algunos ajustes"
Bloque "Navegación"
    Botón "Cancelar"
    Botón "Anterior"
    Botón "Siguiente"
