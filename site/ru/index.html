Попробуй CLI

Мы сейчас упаковываем ImagineUI в консольную программу. Пока можете попробовать альфа-версию, скачав и собрав проект с GitHub.

# Сначала нужно глобально поставить Puppeteer npm install -g puppeteer git clone git@github.com:imagineui/imagineui.git cd imagineui yarn yarn start --input=%full-path-to-.scene% --output=%full-path-to-.png%
